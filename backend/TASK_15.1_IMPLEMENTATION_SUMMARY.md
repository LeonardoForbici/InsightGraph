### 1. Added `count_vulnerabilities()` Helper Function
**File:** `backend/fragility_calculator.py`

Added `_count_vulnerabilities()` method that queries Neo4j for SecurityIssue nodes linked to the target node via HAS_VULNERABILITY relationships:

```python
def _count_vulnerabilities(self, node_key: str) -> int:
    """
    Count SecurityIssue nodes linked to this entity via HAS_VULNERABILITY relationship.
    
    Requirements: 10.5
    """
    if self._neo4j and getattr(self._neo4j, "is_connected", False):
        try:
            query = """
            MATCH (e:Entity {namespace_key: $key})-[:HAS_VULNERABILITY]->(s:SecurityIssue)
            RETURN count(s) AS vuln_count
            """
            rows = self._neo4j.graph.run(query, key=node_key).data()
            if rows and rows[0].get("vuln_count") is not None:
                return int(rows[0]["vuln_count"])
            return 0
        except Exception as e:
            logger.warning("FragilityCalculator._count_vulnerabilities Neo4j error: %s", e)
    
    # Memory fallback - no vulnerability data in memory
    return 0
```

### 2. Modified `calculate_fragility_score()` to Include Vulnerability Factor
**File:** `backend/fragility_calculator.py`

Updated `_compute_score()` method to include vulnerability parameter:

```python
def _compute_score(
    self,
    dependents: int,
    depth: int,
    complexity: int,
    hops_to_frontend: int = 0,
    vulnerabilities: int = 0,
) -> float:
    """
    Weighted fragility formula clamped to [0, 100].

    score = min(100,
        (dependents_count / MAX_DEPENDENTS) * 40 +
        (graph_depth / MAX_DEPTH) * 30 +
        (cyclomatic_complexity / MAX_COMPLEXITY) * 30 +
        max(0, (hops_to_frontend - 3)) * 5 +
        min(vulnerabilities * 10, 30)
    )

    Requirements: 4.1, 4.2, 10.5
    """
    raw = (
        (dependents / MAX_DEPENDENTS) * 40
        + (depth / MAX_DEPTH) * 30
        + (complexity / MAX_COMPLEXITY) * 30
        + max(0, hops_to_frontend - 3) * 5
        + min(vulnerabilities * 10, 30)
    )
    return float(min(100.0, max(0.0, raw)))
```

**Vulnerability Weighting:**
- +10 points per vulnerability
- Capped at 30 points maximum (3+ vulnerabilities)
